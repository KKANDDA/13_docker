서현준 단편 소설

"사람은 엉덩이로 똥을 싸지만, 거미는 실을 뽑습니다." 유투브로 곤충학자가 말하는 거미 관련 영상을 봤는데, 놀랍더군요. 거미가 곤충이 아니라는 이야기는 예상했지만, 거미에게 비행 능력이 있다는 이야기는 신비했습니다. 날개가 있는 것도 아닌데 말이죠. 일단 비행에 대한 정의가 필요할 듯 한데요. 무엇을 비행이라고 말할 수 있을지, 어디까지를 비행이라고 할 수 있을지는 모르겠지만, 제 생각에 상공 5000m까지 날아 올랐다면 일단은 비행이라고 말해도 되지 않을까 싶습니다. 이제 거미가 어떻게 비행을 시작을 하는지 공개할 차례군요. 거미는 엉덩이.. 아, 아니.. 거미줄로 날아 오릅니다. 새들이 날개짓으로 바람을 일으킨다면, 거미는 그냥 바람을 타는 거죠. 마치.. 서핑이라고 할까요?
